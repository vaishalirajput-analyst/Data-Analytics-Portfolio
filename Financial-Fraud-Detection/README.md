# Capstone Project: Financial Fraud Detection Pipeline

## 📌 Project Overview
Engineered an end-to-end automated data pipeline using Python and Tableau to ingest, clean, and analyze high-volume financial transaction records. The core objective of this project was to identify anomaly patterns, flag potential fraudulent transactions, and present actionable risk insights through data visualization frameworks.

## 🛠️ Tech Stack & Skills Demonstrated
* **Languages:** Python (Pandas, NumPy, Matplotlib, Seaborn)
* **Business Intelligence:** Tableau
* **Methodologies:** Exploratory Data Analysis (EDA), Data Cleansing, Data Manipulation, Outlier & Anomaly Detection, Time-Series Analysis

## 🚀 Key Technical Executions
1. **Data Ingestion & Hygiene:** Built a Python workflow using Pandas to handle raw transaction data, isolating structural deficiencies, managing missing values, and executing data cleansing.
2. **Exploratory Data Analysis (EDA):** Leveraged NumPy and statistical modeling to run descriptive statistics, successfully identifying anomalous outliers and high-risk spending velocities.
3. **Interactive Dashboarding:** Developed a dynamic Tableau BI dashboard incorporating calculated fields and granular filters to map geospatial risk distributions and time-series fraud indicators for stakeholder monitoring.
